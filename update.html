üìñ How to Update Your Portfolio

üì∏ Step 1: Upload Images/Files to GitHub

  1. For projects: Upload images to images/projects/ folder
  2. For image certificates: Upload to images/certificates/ folder
  3. For PDF certificates:
    • Upload PDF to files/certificates/ folder
    • Take screenshot of first page
    • Upload screenshot to images/certificates/ folder

üîó Step 2: Get File URLs

  1. Go to your file on GitHub
  2. Click "Download" or "View Raw"
  3. Copy the URL from address bar
  4. For GitHub raw URLs: Use format:
    https://raw.githubusercontent.com/YOUR_USERNAME/portfolio/main/images/.../file.jpg

üìù Step 3: Fill the Form

  1. Select correct tab (Project or Certificate)
  2. Fill all required fields (*)
  3. Paste your image/file URLs
  4. Click "Preview" to check
  5. Click "Generate JSON Code"

üìÑ Step 4: Update JSON Files

  1. Copy the generated JSON code
  2. Go to your data/ folder on GitHub
  3. Edit projects.json or certificates.json
  4. Add the new item to the array (between [ ])
  5. Make sure items are separated by commas
  6. Save and commit changes

üéâ Step 5: View Your Portfolio

  1. Your portfolio will automatically show new items
  2. Refresh your portfolio page (index.html)
  3. New projects/certificates will appear!

üí° Tips

  • Keep image sizes reasonable (under 1MB)
  • Use descriptive names for files
  • Test URLs by opening them in browser
  • Always preview before adding
