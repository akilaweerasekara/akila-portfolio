üìñ How to Update Your Portfolio

                
                

                    
Step 1: Upload Images to GitHub

                    
    
                        
  1. Go to your GitHub repository
    2. 
                        
  2. Click "Add file" ‚Üí "Upload files"
    3. 
                        
  3. Drag & drop your project/certificate images
    4. 
                        
  4. Upload to: images/projects/ or images/certificates/
    5. 
                        
  5. Commit changes
    6. 
                    

                    
                    
Step 2: Get Image URL

                    
    
                        
  1. Click on the uploaded image in GitHub
    2. 
                        
  2. Click "Download" button
    3. 
                        
  3. Copy the URL from browser address bar
    4. 
                        
  4. Remove "?raw=true" from the end if present
    5. 
                    

                    
                    
Step 3: Fill the Form

                    
    
                        
  1. Fill in all details in the form above
    2. 
                        
  2. Paste the image URL
    3. 
                        
  3. Click "Preview" to check
    4. 
                        
  4. Click "Add to Portfolio"
    5. 
                    

                    
                    
Step 4: Update JSON Files

                    
    
                        
  1. Copy the generated JSON code
    2. 
                        
  2. Go to data/projects.json or data/certificates.json
    3. 
                        
  3. Paste the new item in the array (between [ ])
    4. 
                        
  4. Save and commit to GitHub
    5. 
                    

                    
                    
Step 5: View Your Portfolio

                    
    
                        
  1. Your portfolio will automatically show new items
    2. 
                        
  2. Refresh your portfolio page to see changes
    3.
