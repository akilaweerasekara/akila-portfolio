Bio Palm Project

Portable Palm-Based Non-Invasive Health Monitoring System

Bio Palm is an innovative portable health monitoring system designed to measure key health parameters non-invasively through the human palm. It integrates multiple sensors including the MAX30105 for heart rate and SpO₂, DS18B20 for body temperature, MQ135 for air quality, and TTP223 touch sensor for activation — all powered by an ESP32 microcontroller.

The system transmits real-time data wirelessly to a custom-built web app via WebSocket. Users can view live readings, receive alerts in patient mode, and generate health reports as PDFs. The web interface was built using modern HTML, CSS, and JavaScript with Firebase authentication support.
